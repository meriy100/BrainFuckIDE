## Configuration

- npm version x.x.x

### Elm install 

Prease view to https://guide.elm-lang.org/install.html

```sh
$ npm install -g elm
```


### elm-live install

```sh
$ npm isntall -g elm-live
```


### Yarn install


```sh 
$ npm install -g yarn
```


### packages


```
$ elm install 
$ yarn install
```


### Run Applicatgion


```
$ npm run elm-live
$ npm run webpack-watch
```
